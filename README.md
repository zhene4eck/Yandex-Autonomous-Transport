# Техническое задание в команду Автономного транспорта

## Этапы выполнения задачи представляют полуавтоматический конвеер

Ряд задач выполняются строго по построенному нижеприведеннорму алгоритму, с использованием встроенных плагинов для процессов оптимизации. В силу ряда повторяющихся процессов подготовки и обработки сновы, в том числе изменений в атрибутивной состовляющей большая часть процесса использует оптимизацию с помощью кода на языке программирования Python. 
| Обрати внимание! |
|:-----------|
| Ряд процессов оптимизируется благодаря плагинам и скриптам, однако каждый этап необходимо контролировать ручной проверкой воизбежание критических ошибок или сбоев программы/скрипта |

1. Автоматическая подготовка основы
2. Предварительная обработка с использованием плагинов для оптимизации
3. Применение SQL запроса для подготовки атрибутивной состовляющий
4. Ручная проверка
